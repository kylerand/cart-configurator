## Common Concerns Addressed

### "This seems like a lot of work"
- **Response:** We only need 10-15 core options to launch. Full catalog can be added over time. Start with your bestsellers.

### "We don't have exact prices yet"
- **Response:** Ballpark estimates are fine for Phase 1. We need relative pricing (X costs more than Y) to build the system. Exact pricing can be updated anytime.

### "What if we offer custom options?"
- **Response:** Define the standard templates. Custom work is handled during quote review, not in the configurator.

### "Our rules are complicated"
- **Response:** Start with the obvious ones (can't have two roofs). Edge cases can be added incrementally.

### "How accurate do labor hours need to be?"
- **Response:** ¬±30 minutes is fine. We're building quotes, not scheduling. Round to nearest 0.5 hours.

   - Build test configurations
   - Verify calculations
   - Check rule enforcement

3. **Soft launch** (2 weeks)
   - Internal use only

   - Gather feedback

4. **Full launch**
   - Customer access
   - Marketing rollout
   - Ongoing refinement
